### keyboard를 이용한 간단한 조작

* <kbd>→</kbd> ,  <kbd>←</kbd> : 정밀 각 조정  

- <kbd>↑</kbd> ,  <kbd>↓</kbd> : 큰 각 조정  

- <kbd>space</kbd>  :  눌렀다 때는경우 누른 시간만큼의 세기로 **hit**
- <kbd>c</kbd>  :  플레이어 변경
